# UniShield-Secure: AI-Driven Zero Trust Knowledge Logistics

##  Project Overview
[cite_start]**UniShield-Secure** is a strategic cybersecurity framework designed for higher education innovation[cite: 1]. [cite_start]It addresses the **$3.8M data breach crisis** in universities by replacing outdated "hub-and-spoke" network architectures with a **Zero Trust Campus** model[cite: 17, 33, 47].

### The Problem
* [cite_start]**Logistics Crisis**: Disconnected heterogeneous information systems conflict with rigorous data privacy requirements[cite: 3, 4].
* [cite_start]**Insider Threats**: Over 60% of organizations face threats from internal personnel leveraging legitimate access[cite: 36, 37].
* [cite_start]**Web Vulnerabilities**: 80% of library flaws are application-level threats like SQL injection and XSS[cite: 30].

##  Key Features
* [cite_start]**Self-Sovereign Identity (SSI)**: Uses Decentralized Identifiers (DIDs) and blockchain for tamper-proof academic credentials[cite: 62, 65].
* [cite_start]**AMD-Powered UEBA**: AI-driven User and Entity Behavior Analytics for real-time anomaly detection with an F1-score of 0.90[cite: 74, 79].
* [cite_start]**Secure RAG**: Retrieval-Augmented Generation for research data using "filter-first" metadata policies to ensure data privacy[cite: 80, 88].
* [cite_start]**Multilingual Support**: Integration with **Param-2** and **Shrutam** for access in 22 regional Indian languages[cite: 103, 105].

##  Technical Stack & AMD Integration
* **AI Engine**: Python, PyTorch, and LlamaIndex optimized for **AMD Ryzen AI** and **ROCm**.
* [cite_start]**Blockchain**: Solidity zkEVM smart contracts for private identity management[cite: 68, 93].
* [cite_start]**Infrastructure**: Edge-AI deployment on low-cost hardware for rural university accessibility[cite: 98, 99].

##  Repository Structure
* [cite_start]`/blockchain`: Solidity contracts for DID/VC implementation.
* [cite_start]`/ai_engine`: Python scripts for UEBA anomaly detection and the RAG pipeline[cite: 73, 93].
* [cite_start]`/frontend`: Multilingual React.js dashboard with integrated Web3.js[cite: 139].
* [cite_start]`/docs`: Technical architecture diagrams and compliance documentation[cite: 139].
